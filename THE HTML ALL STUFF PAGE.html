Lists

Unordered

Unordered Lists: Use ul tag to begin and end list, li tag for each item of the list.

Normal: (Same as ul style='list-style-type:disc' (default))

  • Item 1
  • Item 2
  • Item 3
  • Item 4

With ul style='list-style-type:circle'

  • Item 1
  • Item 2
  • Item 3
  • Item 4

With ul style='list-style-type:square'

  • Item 1
  • Item 2
  • Item 3
  • Item 4

With ul style='list-style-type:none'

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Ordered

Ordered Lists: Use ol tag to begin and end list, li tag for each item of the list.

Normal: (Same as ol type="1"(default))

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

With ol type="A" //(Uppercase letters)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

With ol type="a" //(Lowercase letters)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

With ol type="I" //(Roman Numbers)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

With ol type="i" //(Lowercase Roman)

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Control list counting with ol start="number to start list with".

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Nested Lists:

Unordered: (use another ul in the list).

  • Item 1
    • Item 2
    • Item 3
  • Item 4

Ordered: (use another ol in the list).

  1. Item 1
    1. Item 2
    2. Item 3
  2. Item 4

Description lists:

Use dl to begin and end list, dt tag for description term, dd tag for description definition. (dd gets indented)

Div

Div (division) tag is a block element (occupies its own line) used to keep stuff in neat blocks. You can add styles to it too (border, border-radius, background-color, etc.)

Use margin: auto with a set width!

Multiple divs:

You can put multiple divs in a page. Each div occupies its own line:

London

Oslo

Divs in same line:

Use display:flex or display:inline-block or display:grid (for display:flex, you need to make a parent div which contains the other divs and give the parent the display property (after that you can add all other divs without it and it will display in the same line)) or float:left/right (left makes them float to left (like image in line with text) so the divs start from left to right)(float:right makes them start from the right).
